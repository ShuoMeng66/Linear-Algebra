# Release Notes

## 当前版本

当前仓库可以视作第一版公开可用版本，已经具备下面这些核心内容：

- 一套完整的 LaTeX 线性代数讲义
- 一份可直接阅读的 PDF 成品
- 一条清晰稳定的五章主线
- 面向零基础和补考同学的例题讲解风格
- 每章更丰富的例题与 `A / B / C` 分级课后习题
- 一个可导入 Cherry Studio 的教材助教
- 配套的 `README`、`CONTRIBUTING`、`LICENSE` 与素材说明

## 本版重点更新

### 讲义部分

- 进一步强化了“从方程组出发”的整体叙述逻辑
- 补充了更多典型三阶矩阵例题
- 重新扩充了每章课后练习
- 将习题分为 `A / B / C` 三档，方便基础训练、常规巩固与综合提升

### 教材助教部分

- Cherry Studio 助教不再只是“讲题”，而是升级成更完整的伴学型助教
- 新增对“证明题某一步为什么成立”的细化解释能力
- 新增对“学生自己写的证明或思路”的批改能力
- 新增对“做到一半做不下去”的提示式引导能力
- 新增对 OCR 文本、凌乱草稿和错题复盘的支持

### 开源仓库部分

- 调整了 GitHub 首页 README 的信息结构
- 明确了快速开始、教材下载、Cherry Studio 导入与本地编译路径
- 同步完善了贡献说明与素材说明

## 适合直接拿来做什么

- 自学入门
- 补考冲刺
- 课程配套讲义
- Cherry Studio 教材助教订阅
- 后续继续扩写成更完整的开源教材

- 补更多“参数题”和“证明题”的系统专题
