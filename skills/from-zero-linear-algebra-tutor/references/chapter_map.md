# 教材章节地图

## 第一章 线性方程组与 Gauss 消元法

核心关键词：

- 齐次与非齐次
- 增广矩阵
- Gauss 消元
- rank
- 主变量与自由变量
- 基础解系
- 特解 + 齐次通解

常见提问：

- 为什么先学齐次方程组
- rank 到底是什么意思
- 为什么非齐次通解一定是特解加齐次通解
- 基础解系怎么从消元结果里写出来

## 第二章 矩阵的基本运算

核心关键词：

- 加法与数乘
- 矩阵乘法
- 转置
- 逆矩阵
- Gauss-Jordan 求逆
- 行列式
- 伴随矩阵
- Cramer 法则

常见提问：

- 矩阵乘法为什么这样定义
- 逆矩阵和解方程有什么关系
- 为什么本书把行列式放到这里才讲
- 三阶矩阵怎么稳妥地求逆

## 第三章 矩阵的相似对角化

核心关键词：

- 特征值
- 特征向量
- 特征空间
- 特征多项式
- 相似对角化
- 不能对角化的情形
- 迹与行列式和特征值的关系

常见提问：

- 特征值为什么会冒出来
- 为什么求特征向量还是回到消元法
- 什么条件下矩阵能相似对角化
- 为什么有些三阶矩阵不能对角化

## 第四章 矩阵的合同对角化

核心关键词：

## 第五章 二次型与正定矩阵

核心关键词：

- 二次型
- 对称矩阵
- 标准形
- 正定、负定、不定
- 顺序主子式判别法
- 正特征值

常见提问：

- 为什么只看对称矩阵就够了
- 二次型怎么化成标准形
- 顺序主子式判别法怎么用
- 正定矩阵为什么特征值全为正

## 全书主线

```text
线性方程组与 Gauss 消元
-> 矩阵基本运算
-> 特征值分解 / 相似对角化
-> 正交矩阵 / 合同对角化
-> 二次型与正定矩阵
```

回答时，尽量把任何局部问题重新接回这条主线。
