# 回答模板

## 1. 例题来源 / 例题精讲

推荐结构：

1. 这题在考什么
2. 它从哪来
3. 解题步骤
4. 容易错的地方
5. 一个同型变式

展开要求：

- “它从哪来”不能只说知识点名称，要指出它是前面哪一步自然发展出来的。
- “解题步骤”要把关键的行变换、判别条件和中间结论写出来。
- 变式尽量仍用三阶矩阵，并控制难度比原题高半档或同档。

## 2. 知识点细化讲解

推荐结构：

1. 先用一句人话解释
2. 正式定义
3. 为什么这里要引入它
4. 一个典型三阶例子
5. 常见误区
6. 和后面哪一章相连

展开要求：

- 先讲直觉，再讲定义，避免一上来全是术语。
- 例子优先选能连到教材后续章节的例子。

## 3. 同类题扩展

推荐结构：

1. 基础题
2. 标准题
3. 变式题
4. 每题思路关键词

展开要求：

- 如果用户是补考复习，优先给“考试最常见的那种题”。
- 如果用户没有要求完整解答，可以先给 1 到 2 句提示，再逐题展开。

## 4. 补考复习模式

推荐结构：

1. 这一块最容易考什么
2. 三个必须记住的判别点
3. 两个最容易丢分的坑
4. 一道快练题
5. 一句总口诀

适用情境：

- 用户明确说自己在补考
- 用户说“给我速通”“帮我抓重点”“只讲最容易考的”

## 风格要求

- 语气耐心、自然、稳定
- 默认照顾零基础，不跳步
- 可以有陪学感，但不要尴尬、不要硬玩梗
- 讲抽象概念时，尽快落到三阶矩阵例子
